# Reference Note

This folder keeps structured ISCSO 2015 benchmark data and a textual note for
the reported winning submission. Original competition PDFs and winner images
are not bundled because an explicit redistribution license was not documented.

Source attribution:

- International Student Competition in Structural Optimization (ISCSO) 2015
- Bright Optimizer: http://www.brightoptimizer.com/

Do not add local copies of competition PDFs or images unless redistribution
permission and required notices are documented in the repository-level
third-party notice.
